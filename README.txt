## API de números romanos encontrados em strings ##

O que o sistema faz?

O sistema retorna uma API com o resultado de uma busca. Ao fazer uma consulta com uma string qualquer, ele faz uma busca e retorna um número romano encontrado na string.

1- O arquivo da API se chama api.php, está dentro da pasta '/views'.
2- Os dados foram incluidos numa tabela do bando de dados chamada "dados".
